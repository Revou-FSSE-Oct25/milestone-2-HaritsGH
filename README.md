# RevoFun Website

## Overview

A mock-up website for a fictional game company. Three games are available to play in the website:

1. Rock, Paper, Scissors
2. Guess the Number
3. Fast Clicker

You can access the website by clicking [here](https://revou-fsse-oct25.github.io/milestone-2-HaritsGH/) or visit<br>
https://revou-fsse-oct25.github.io/milestone-2-HaritsGH/

## Features Implemented

### Universal Features
These things are implemented to each games available in this website:
- A way to get to the game-browsing page.
- A way to get to the home/landing page.
- `Play again` button.

### Game-specific Features

1. Rock, Paper, Scissors
   - Click on one of the choices to play the game.

2. Guess the Number
   - Hints of guess higher/lower are given when you guessed wrong.
   - Pressing `Enter` key while typing the number inside the designated space will submit your answer.
  
3. Fast Clicker
   - You can only increase the number of clicks by clicking the designated button.
   - Game will automatically end after the timer runs out.
